# SillyTavern ChatVault · 聊天档案

> 一个 SillyTavern 第三方插件 —— 解决侧边栏只显示最近 15 条聊天的限制，把所有角色、所有聊天都收进一个可搜索、可分类、可收藏、可导入导出的全局档案柜里。

**当前版本：v0.2.4**

---

## 功能一览

- **四视图分类**
  - **最近** — 按最后一条消息时间倒序，跨角色看你最近在和谁聊
  - **按角色** — 折叠分组，每个角色下展开看自己所有聊天
  - **收藏** — 只显示打了星的聊天
  - **当前角色** — 只看当前选中角色的所有聊天，顶部带「新建聊天」「导入」按钮
- **酒馆式聊天卡片**
  - 第一行：标题大字（自定义优先；没设就显示文件名）
  - 第二行：角色名小字（在「按角色」/「当前角色」tab 自动隐藏，避免重复）
  - 元信息：消息数 · 大小 · 相对时间 · 标签
  - 酒馆式最后一条消息预览，懒加载
- **管理操作**（鼠标移上去显现）
  - 收藏 / 取消收藏
  - 编辑：自定义标题 · 标签 · 重命名文件
  - 删除（带确认）
  - 一键跳转：自动切换角色 + 打开该聊天
- **导出**（在编辑窗口里，可折叠）
  - **jsonl** — 酒馆原生格式，可拖回任何角色或者发给别人
  - **txt** — 人类可读 `【角色】消息内容` 格式，带标题头
  - 导出 txt 时可剥离标签内容（默认开启）：
    - `<thinking>…</thinking>`、`<think>…</think>`、HTML 注释
    - 自定义标签对（按字面量匹配，可加任意多对）
- **导入**（当前角色 tab 顶部）
  - 选 `.jsonl` 文件 → 二次确认 → 自动加入该角色聊天列表
- **搜索** — 角色名 / 标题 / 文件名 / 标签，实时过滤
- **分页** — 每页 50 条
- **三套配色** — 夜间 Dark / 白底 Light / 咖啡 Coffee，扩展设置里实时切换
- **总开关** — 不想用了一键关闭入口按钮，无需卸载
- **手机端友好** — 弹窗自适应、内容可滚、显式关闭按钮、滑块开关

## 安装

1. 打开 SillyTavern → 右上角 **扩展** 图标 → **下载扩展和资源**
2. 粘贴仓库地址：
   ```
   https://github.com/shanye5593/SillyTavern-ChatVault
   ```
3. 点 **Save**，刷新页面

后续会自动检测更新（manifest 里 `auto_update: true`）。

---

## 使用

安装后在 **扩展菜单** 里会出现 **「聊天档案」** 入口，点开即可。

**面板内：**
- 顶部四个 tab 切换 最近 / 按角色 / 收藏 / 当前角色
- 搜索框：角色名、聊天标题、文件名、标签都能搜
- 鼠标移到一行：右侧出现 收藏 / 编辑 / 删除 / 跳转 四个操作
- 点跳转图标立刻切换角色并打开那条聊天
- 编辑窗口里可折叠展开「导出聊天」区块，选 jsonl 或 txt 下载
- 当前角色 tab 顶部「导入」按钮：选本地 jsonl 文件加入该角色
- 按 **ESC** 或点遮罩 / 右上角 × 关闭

**配色 / 总开关：**
- 进 SillyTavern 顶部菜单 **扩展** → 找到 **聊天档案 (ChatVault)** 抽屉
- 切换配色方案，已打开的面板会立即变色
- 关掉「启用入口按钮」就把入口隐藏

## 数据存储

- 收藏 / 自定义标题 / 标签：存在浏览器 `localStorage`（键：`st-chatvault-meta`）
- 配色 / 总开关 / 剥离设置：`localStorage`（键：`st-chatvault-settings`）
- **不上传任何数据到外部服务器**，所有调用都是酒馆本地 API

---

## 兼容性

- SillyTavern **1.12+**（release 分支）
- 只使用酒馆公开 HTTP API（`/api/characters/all`、`/api/characters/chats`、`/api/chats/get`、`/api/chats/rename`、`/api/chats/delete`、`/api/chats/import`、`/api/chats/save`）
- 不改核心代码、不依赖其他插件
