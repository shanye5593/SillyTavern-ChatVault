# SillyTavern ChatVault · 聊天档案

> 一个 SillyTavern 第三方插件 —— 解决侧边栏只显示最近 15 条聊天的限制，把所有角色、所有聊天都收进一个可搜索、可分类、可收藏的全局档案柜里。

**当前版本：v0.2.0**

---

## 功能一览

- **三视图分类**
  - **最近** — 按最后一条消息时间倒序，跨角色看你最近在和谁聊
  - **按角色** — 折叠分组，每个角色下展开看自己所有聊天
  - **收藏** — 只显示打了 ⭐ 的聊天
- **酒馆式聊天卡片**
  - 头像 · 角色名 · 自定义标题 · 原始文件名 · 消息数 · 大小 · 相对时间 · 标签 · 最后一条消息预览
  - 列表懒加载预览，滚动到才请求，不卡
- **管理操作**
  - ⭐ 收藏 / 取消收藏
  - ✏️ 编辑：自定义标题（不动文件名，只改显示）/ 标签 / 重命名文件
  - 🗑️ 删除（带确认）
  - ↗️ 一键跳转：自动切换角色 + 打开该聊天
- **搜索** — 角色名 / 标题 / 文件名 / 标签，实时过滤
- **分页** — 每页 50 条，不会被几百条聊天拖崩
- **三套配色** — 夜间 Dark / 白底 Light / 咖啡 Coffee，扩展设置里实时切换
- **总开关** — 不想用了一键关闭入口按钮，无需卸载

## 安装

1. 打开 SillyTavern → 右上角 **扩展** 图标 → **下载扩展和资源**
2. 粘贴仓库地址：
   ```
   https://github.com/shanye5593/SillyTavern-ChatVault
   ```
3. 点 **Save**，刷新页面

后续会自动检测更新（manifest 里 `auto_update: true`）。

---

## 使用

安装后在 **扩展菜单** 里会出现 **「聊天档案」** 入口，点开即可。

**面板内：**
- 顶部三个 tab 切换 最近 / 按角色 / 收藏
- 搜索框：角色名、聊天标题、文件名、标签都能搜
- 鼠标移到一行：右侧出现 ⭐ ✏️ 🗑️ ↗️ 四个操作
- 点 ↗️ 立刻跳到那条聊天
- 按 **ESC** 或点遮罩关闭

**配色 / 总开关：**
- 进 SillyTavern 顶部菜单 **扩展** → 找到 **聊天档案 (ChatVault)** 抽屉
- 切换配色方案，已打开的面板会立即变色
- 关掉「启用入口按钮」就把入口隐藏

---

## 数据存储

- 收藏 / 自定义标题 / 标签：存在浏览器 `localStorage`（键：`st-chatvault-meta`）
- 配色 / 总开关：`localStorage`（键：`st-chatvault-settings`）
- **不上传任何数据到外部服务器**，所有调用都是酒馆本地 API

---

## 兼容性

- SillyTavern **1.12+**（release 分支）
- 只使用酒馆公开 HTTP API（`/api/characters/all`、`/api/characters/chats`、`/api/chats/get`、`/api/chats/rename`、`/api/chats/delete`）
- 不改核心代码、不依赖其他插件
