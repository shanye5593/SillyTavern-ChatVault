# SillyTavern ChatVault 聊天记录管理器

一个 SillyTavern 第三方插件，解决侧边栏只显示 15 条最近聊天的限制，提供**跨角色**的聊天档案总览、搜索和管理。

## 功能

- 📚 **全局总览** — 一个面板查看所有角色的所有聊天档案
- 🔍 **快速搜索** — 按角色名或聊天名实时过滤
- ⭐ **收藏** — 标记重要聊天（保存在浏览器本地）
- ✏️ **重命名 / 🗑️ 删除** — 直接在面板内管理聊天文件
- 🚀 **一键跳转** — 点击任意聊天直接切换角色并打开

## 安装

1. 打开 SillyTavern
2. 点击右上角 **扩展** 图标 → **下载扩展和资源**
3. 输入仓库地址：
   ```
   https://github.com/shanye5593/SillyTavern-ChatVault
   ```
4. 点击 **Save**，安装完成后刷新页面

## 使用

安装后会在 **扩展菜单** 增加一个 **「聊天记录管理器」** 入口，点击即可打开。

- **单击聊天行** → 显示操作按钮（打开 / 重命名 / 删除）
- **双击聊天名** → 直接跳转到该聊天
- **点击 ☆** → 收藏 / 取消收藏
- **顶部"仅看收藏"** → 只显示收藏的聊天
- **ESC** → 关闭面板

## 兼容性

- SillyTavern 1.12+ release 分支
- 仅使用酒馆官方公开 API，不修改核心代码

## 数据存储

- 收藏信息存储在浏览器 `localStorage`（key: `st-chatvault-meta`）
- 不上传任何数据到外部服务器
